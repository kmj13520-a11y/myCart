# 🌿 myCart — 친환경 감성 쇼핑 카트

자연을 담은 미니멀 쇼핑 경험, **myCart**  
React로 제작한 온라인 쇼핑 카트 웹앱으로, 를 적용해 부드럽고 편안한 쇼핑 UI를 구현했습니다.

---

## ✨ 주요 기능 (Features)

### 🛒 쇼핑 카트 기능

- 상품 목록 → 상품 상세 → 장바구니까지 자연스러운 흐름
- 구매 수량 증가/감소, 총 금액 자동 계산
- 개별 상품 삭제 기능
- React 상태 관리 기반의 즉각적인 UI 반응

### 📱 반응형 레이아웃

- 모바일, 태블릿, 데스크톱 화면 대응
- 화면 크기에 따라 상품 카드/레이아웃 유연하게 재배치
